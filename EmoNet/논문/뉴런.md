<뉴런의 데이터들>
1. 입력 전위: K
2. 자극 벡터: stim_vec
3. 기억 전위: K_remem
4. 



<뉴런의 공통 기능 **중요: 기능의 발동 조건: input_K 가 $K_{임계}$ 보다 클때**>
- 기억 시퀀스 실행
- stim_vec의 세로토닌 양에 비례해서 연결 된 시냅스 삭제
- stim_vec의 도파민 양에 비례해서 시냅스 추가

<억제성 뉴런 특수 기능>
1. stim_vec 평준화(K 비례)

<흥분성 뉴런 특수 기능>
1. stim_vec 비평준화(K 비례)

<조절성 뉴런 특수 기능>
1. stim_vec의 멜라토닌 양에 비례해서 랜덤 노드 드롭아웃(다음 사이클에서 되살아 남)
2. stim_vec의 노르에피네프린 양에 비례해서 $K_{임계}$, $K_{remem}$ 감소(타임 스텝에 따라 서서히 다시 원래 상태로 돌아감)




<기억 시퀀스>
- K_remem이 넘어서면 memory 저장
- memory = {stim_vec, K, Input, Cycle Number}
- Cycle Number에 따라 감쇠 적용(일정 값 이하로 떨어지면 삭제) 후 K로 가중 합 계산 및 현재 stim_vec에 가중합
- 각각 모든 memory에 대하여 현재 stim_vec과의 유사도에 따라 K 증가
